Abstract

Luminara bridges the gap between astrophysics and artificial intelligence by providing an interpretable, visual, and interactive platform for exoplanet detection. The system leverages high-precision stellar data, advanced signal preprocessing, and a custom deep learning model to identify potential exoplanet transits. Unlike black-box AI models, Luminara emphasizes interpretability through saliency maps, confidence scoring, and calibrated probabilities, making it suitable for both researchers and educational purposes.

The primary goal is to automate and simplify the process of analyzing flux data from stars while providing insights that human researchers can understand and trust.

Features & Pipeline

1. Stellar Data Retrieval

Input a TIC ID and Luminara fetches contextual stellar metadata such as RA, DEC, magnitude, and other catalog information. This provides essential context for interpreting flux data and potential exoplanet candidates.

2. Flux Time Series Processing

Uses Python's lightkurve library to retrieve flux versus time data. Preprocessing includes:

  • Median Absolute Deviation (MAD) for outlier removal
  • Normalization and scaling for consistent model input
  • Handling missing or corrupted data points for robust analysis
3. Custom Deep Learning Model

Luminara uses a PyTorch-based CNN trained on the Kepler mission dataset. The model is tailored for sequential astrophysical signals and includes:

  • Saliency Map Generation: Highlights regions in flux data corresponding to possible exoplanet transits, improving interpretability.
  • Region-specific Confidence Scoring: Assigns localized probabilities to candidate regions.
  • Temperature Scaling: Applies softmax calibration to produce reliable confidence estimates, reducing overconfident predictions.
4. Interactive Visualization Dashboard

Luminara provides intuitive visualizations to make the results accessible and actionable:

  • Flux vs. Time plots with saliency overlays
  • Highlighted candidate regions for potential exoplanet transits
  • Confidence metrics summarizing model predictions
  • Downloadable plots and summaries for further analysis
5.Highlights
  • End-to-end scientific AI pipeline from raw stellar data to interpretable results
  • Integration of advanced preprocessing, CNN, and calibrated confidence techniques
  • Strong focus on interpretability, making the system trustworthy for domain experts
  • Interactive dashboard demonstrates full-stack AI capability: Python backend + visualization + user interaction
